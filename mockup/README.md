# Argu 목업 디자인

이 디렉토리는 Argu 논쟁 플랫폼의 목업 디자인 파일들을 포함합니다.

## 📁 디렉토리 구조

```
mockup/
├── index.html              # 메인 페이지
├── argu-list.html          # 논쟁 목록 페이지
├── argu-detail.html        # 논쟁 상세 페이지
├── argu-create.html        # 논쟁 작성 페이지
├── argu-edit.html          # 논쟁 수정 페이지
├── categories.html         # 카테고리 목록 페이지
├── category-detail.html    # 카테고리별 논쟁 페이지
├── search.html             # 검색 페이지
├── user-profile.html       # 사용자 프로필 페이지
├── my-page.html            # 마이페이지 (대시보드)
├── auth/
│   ├── login.html          # 로그인 페이지
│   └── register.html       # 회원가입 페이지
├── admin/
│   ├── index.html          # 관리자 대시보드
│   ├── users.html          # 회원 관리 페이지
│   ├── argu.html           # 논쟁 관리 페이지
│   ├── comments.html       # 댓글 관리 페이지
│   ├── categories.html     # 카테고리 관리 페이지
│   ├── reports.html        # 신고 관리 페이지
│   ├── statistics.html     # 통계 및 분석 페이지
│   ├── settings.html       # 시스템 설정 페이지
│   ├── admins.html         # 관리자 관리 페이지
│   └── login.html          # 관리자 로그인 페이지
└── css/
    └── style.css           # 공통 스타일시트
```

## 🎨 사용자 사이트 페이지

### 1. 메인 페이지 (index.html)
- 검색 기능
- 인기 논쟁 미리보기
- 최신 논쟁 목록
- 카테고리별 미리보기
- 랭킹 시스템 (인기 논쟁, 인기 사용자)

### 2. 논쟁 목록 (argu-list.html)
- 전체 논쟁 목록
- 필터링 (카테고리, 상태, 정렬)
- 논쟁 카드 형태의 목록 표시
- 페이지네이션

### 3. 논쟁 상세 (argu-detail.html)
- 논쟁 상세 정보
- 찬성/반대 투표 섹션
- 댓글 시스템 (대댓글 지원)
- 실시간 채팅 (논쟁 기간 동안)
- 좋아요/북마크 기능

### 4. 논쟁 작성 (argu-create.html)
- 논쟁 작성 폼
- 카테고리 선택
- 찬성/반대 입장 설명
- 논쟁 기간 설정
- 추가 설정 옵션

### 5. 로그인/회원가입 (auth/)
- 로그인 페이지
- 회원가입 페이지
- 소셜 로그인 버튼 (추후 지원 예정)
- 약관 동의

### 6. 마이페이지 (my-page.html)
- 사용자 프로필
- 통계 요약
- 인기 논쟁 TOP 3
- 최근 활동 내역
- 사이드바 네비게이션

## 🔧 관리자 사이트 페이지

### 1. 관리자 대시보드 (admin/index.html)
- 전체 통계 카드
- 최근 가입 회원
- 인기 논쟁 TOP 5
- 미처리 신고 목록

### 2. 회원 관리 (admin/users.html)
- 회원 목록 테이블
- 검색 및 필터링
- 회원 상태 관리 (정상/정지/탈퇴)
- 회원 상세 및 작업 버튼

## 🎨 디자인 특징

### 색상 테마
- Primary: #6366f1 (인디고)
- Secondary: #8b5cf6 (보라)
- Success: #10b981 (녹색)
- Danger: #ef4444 (빨강)
- Warning: #f59e0b (주황)

### 반응형 디자인
- 모바일, 태블릿, 데스크톱 지원
- 그리드 레이아웃 사용
- 미디어 쿼리 적용

### UI 컴포넌트
- 카드 디자인
- 버튼 스타일
- 폼 요소
- 테이블
- 상태 배지
- 페이지네이션

## 📝 사용 방법

1. 브라우저에서 `index.html` 파일을 열어 메인 페이지를 확인합니다.
2. 각 페이지의 링크를 통해 다른 페이지로 이동할 수 있습니다.
3. 관리자 페이지는 `admin/` 디렉토리 내에 있습니다.
